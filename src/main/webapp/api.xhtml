API REST - Lista de Resources

Lista de Resources

Base URL

http://localhost:8080/consulta/api/

Formato JSON

Nossa API Rest retorna e recebe o formato JSON.

Usuários

  • GET /usuarios

    Lista todos os usuários

  • GET /usuarios/{id}

    Lista um usuário específico

  • PUT /usuarios

    Atualiza um usuário específico. Exemplo JSON: { "id": 1, "login": "demo", "nome": "Usuário DEMO", "email": "demo@google.com", "senha": "pass" }

  • POST /usuarios

    Adiciona um usuário específico. Exemplo JSON: { "login": "demo", "nome": "Usuário DEMO", "email": "demo@google.com", "senha": "pass" }

  • DELETE /usuarios/{id}

    Remove um usuário específico

  • JSON Exemplo:

    { "id": 1, "login": "demo", "nome": "Usuário DEMO", "email": "demo@google.com", "senha": "pass" }

  • POST /pacientes

    Adiciona um paciente específico. Exemplo JSON: { "nome": "Nome do Paciente", "sobrenome": "Sobrenome do Paciente", "email": "paciente@google.com" }

  • DELETE /pacientes/{id}

    Remove um paciente específico

  • JSON Exemplo

    { "id": 1, "nome": "Nome do Paciente", "sobrenome": "Sobrenome do Paciente", "email": "paciente@google.com" }

Médicos

  • GET /medicos

    Lista todos os médicos

  • GET /medicos/{id}

    Lista um médico específico

  • PUT /medicos

    Atualiza um médico específico Exemplo JSON: { "id": 1, "nome": "Pedro", "sobrenome": "Junior", "especialidade": { "id": 1 } }

  • POST /medicos

    Adiciona um médico específico. Exemplo JSON: { "nome": "Pedro", "sobrenome": "Junior", "especialidade": { "id": 1 } }

  • DELETE /medicos/{id}

    Remove um medico específico

  • JSON Exemplo

    { "id": 2, "nome": "Nome do Médico", "sobrenome": "Sobrenome do Médico", "especialidade": { "id": 1, "nome": "Nome da especialidade médica" } }

Consultas

  • GET /consultas

    Lista todas as consultas médicas

  • GET /consultas/{id}

    Lista uma consulta médica específica

  • PUT /consultas

    Atualiza uma consulta médica específica. Exemplo do JSON: { "id": 1, "medico": { "id": 1 }, "paciente": { "id": 1 }, "dataConsulta": "Oct 10, 2015 12:00:00 AM" }

  • POST /consultas

    Adiciona uma consulta médica específica. Exemplo do JSON: { "medico": { "id": 1 }, "paciente": { "id": 1 }, "dataConsulta": "Oct 10, 2015 12:00:00 AM" }

  • DELETE /consultas/{id}

    Remove uma consulta médica específica

  • JSON Exemplo

    { "id": 1, "medico": { "id": 1, "nome": "Nome do Médico ", "sobrenome": "Sobreome do Médico", "especialidade": { "id": 1, "nome": "Nome da especialidade médica" } }, "paciente": { "id": 1, "nome": "Nome do Paciente", "sobrenome": "Sobrenome do Paciente", "email": "paciente@google.com" }, "dataConsulta": "Oct 10, 2015 12:00:00 AM" }
